# Library Magic

Library Magic is an incremental game built around magical books, where each opened book contains a focused mini-game that feeds the shared library progression.

## Language

**Mana**:
The shared resource used by the whole library. Mana is generated by the Grimoire de Mana and can be spent to unlock books or buy upgrades.
_Avoid_: Energy, crystals

**Book Mini-Game**:
A mini-game contained inside an opened book page. It should stay visually inside the book overlay and keep controls simple enough to scale toward multiple simultaneous books.
_Avoid_: Screen, mode, app

**Grimoire de Mana**:
The first book mini-game. It generates only Mana through a clickable mana orb.
_Avoid_: Mana crystal, crystal book

**Livre du Serpent**:
The snake-themed book mini-game. It generates Ecailles and Mana through a simple snake interaction.
_Avoid_: Snake screen, snake app

**Ecailles**:
The unique resource produced by the Livre du Serpent.
_Avoid_: Scales, snake points
